# World Day Of 🌍

A simple, lightweight desktop application built with **Python** and **Tkinter** that allows users to quickly discover which "World Days" are celebrated yesterday, today, and tomorrow.

---

## 🛠️ Prerequisites

Before running the application, ensure you have Python installed on your system, along with the **Pillow** library for handling images and icons.

1. **Python 3.x** (Preferably 3.12)
2. **Pillow (PIL):** (You can install it using `pip install pillow`)
